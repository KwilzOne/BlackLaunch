## Description

### About the Project
BlackLaunch is a minimalist, fast, and resource-efficient launcher for Minecraft, designed for users who prefer a clean and uncluttered interface.

### Project Goals and Motivation
* **Modern Performance:** Most existing launchers are either outdated or built on heavy web frameworks, resulting in high resource consumption. BlackLaunch aims to be lightweight and real fast.
* **No Adware:** The launcher contains no advertisements, recommendations, or sponsored content.
* **Streamlined UI:** Eliminated bloated menus and unnecessary settings, keeping only the essential options required by the average user.

---

## Описание

### О проекте
BlackLaunch — это минималистичный, быстрый и нетребовательный к ресурсам лаунчер для Minecraft, созданный для пользователей, которые предпочитают чистый интерфейс без лишних элементов.

### Цели и мотивация проекта
* **Современная производительность:** Большинство существующих лаунчеров либо устарели, либо построены на тяжелых веб-фреймворках, что приводит к высокому потреблению ресурсов. BlackLaunch разработан легким и реально быстрым.
* **Отсутствие рекламы:** Лаунчер не содержит рекламных баннеров, рекомендаций и партнерского контента.
* **Оптимизированный интерфейс:** Избавились от перегруженных меню и лишних настроек, оставив только важные параметры, необходимые среднестатистическому пользователю.
